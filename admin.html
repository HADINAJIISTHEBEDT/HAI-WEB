Run Web / Stop Web

In the project folder HAI WEB on your Desktop:

  • run-web.bat — starts the site and opens http://localhost:5500
  • stop-web.bat — stops the site
  • start-web.bat — same as run-web.bat

After login you edit on index.html?admin=1 (live site), not this login page.

Terminal: npm run run or .\run-web.bat · Stop: .\stop-web.bat
